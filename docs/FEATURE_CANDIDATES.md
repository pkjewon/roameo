# 추가 기능 후보 리스트

학생들은 아래 기능 중 선택해서 구현합니다. 여러 개를 구현해도 되지만, 많이 넣는 것보다 **제대로 동작하고 사용자가 이해하기 쉬운 기능**이 더 높은 평가를 받습니다.

## 난이도 하

| 기능 | 간단 설명 |
|---|---|
| 다크모드 | 버튼을 누르면 화면 테마가 어둡게 바뀝니다. |
| 랜덤 추천 | 버튼을 누르면 카드 목록 중 하나를 랜덤으로 추천합니다. |
| 카운트다운 | 특정 날짜나 시간까지 남은 시간을 보여줍니다. |
| 맨 위로 이동 버튼 | 버튼을 누르면 페이지 최상단으로 이동합니다. |
| 카드 강조 효과 | 카드에 마우스를 올리거나 클릭했을 때 강조 효과를 줍니다. |

## 난이도 중

| 기능 | 간단 설명 |
|---|---|
| 검색 기능 | 입력한 키워드가 포함된 카드만 보여줍니다. |
| 카테고리 필터 | 선택한 카테고리에 해당하는 카드만 보여줍니다. |
| 정렬 기능 | 카드 목록을 이름순, 카테고리순 등으로 정렬합니다. |
| 좋아요/찜하기 | 사용자가 마음에 드는 카드에 표시를 남깁니다. |
| 결과 없음 메시지 | 검색/필터 결과가 없을 때 안내 문구를 보여줍니다. |

## 난이도 상

| 기능 | 간단 설명 |
|---|---|
| localStorage 저장 | 다크모드, 찜하기, 투표 등 사용자의 선택을 새로고침 후에도 유지합니다. |
| 신청 폼 검증 | 이름, 학과, 관심 분야 등 입력값을 검사하고 오류 메시지를 보여줍니다. |
| 투표 기능 | 선택지를 클릭하면 투표 결과가 화면에 반영됩니다. |
| 찜한 항목만 보기 | 사용자가 찜한 카드만 따로 모아서 보여줍니다. |
| 복합 검색/필터 | 검색어와 카테고리 필터를 동시에 적용합니다. |
