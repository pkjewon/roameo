# Service Update Starter

심화 수업용 스켈레톤 웹서비스입니다.

## 실행 방법

`index.html` 파일을 브라우저로 열면 바로 실행됩니다.

## 파일 구조

```text
service-update-starter-minimal/
├── index.html
├── css/
│   └── style.css
├── js/
│   ├── data.js
│   └── app.js
└── docs/
    └── FEATURE_CANDIDATES.md
```

## 학생 작업 방향

- `index.html`의 기능 추가 영역에 필요한 UI를 배치합니다.
- `css/style.css`에 선택한 기능의 디자인을 추가합니다.
- `js/app.js`에 기능 동작을 구현합니다.
- `docs/FEATURE_CANDIDATES.md`에서 구현할 기능을 선택합니다.
